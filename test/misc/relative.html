This test shows the effects of inserting an absolute div inside a relative div before an element in the DOM. Different configuration are tested, enlarge your window to at least 1000 pixels to see the example before and after the div insertion at the same level. This test exemplifies the techniques used by AXEL layout manager to position the lenses in either 'absolute' or 'inline' mode.
