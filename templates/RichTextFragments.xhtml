XTiger rich text fragments component library

This template contains XTiger XML component(s) that you may copy inside your templates. It also shows how to use some of the plugins available. The common goal of these components and plugins is to edit what we call rich text fragments, meaning not really semantically-structured content but more visually-structured content (styled text, paragraphs, lists, sublists, etc.). The main difficulty in that case is to play around with HTML and CSS to get an acceptable UI with AXEL.
